# Research

本目录承载 Hshare Lab v2 的研究与审计输出。

## 子目录建议

- `Research/Audits/`：DQA、semantic verification、field audit
- `Research/Validation/`：golden sample 验真与对照实验
- `Research/Reports/`：阶段性研究报告
- `Research/Notes/`：临时研究笔记

## 当前规则

- 所有研究结论必须说明基于哪一层数据
- `candidate cleaned` 与 `verified` 的结论不得混用
- legacy 报告不可直接视为 v2 事实
